## Upload Issues

<Steps>
  <Step title="Check File Format" icon="file">
    Ensure your file is PDF, JPG, or PNG and under size limits.
  </Step>
  <Step title="Verify Content" icon="eye">
    Make sure the syllabus text is clear and legible.
  </Step>
  <Step title="Retry Upload" icon="refresh-cw">
    Try uploading again after checking the above.
  </Step>
  <Step title="Contact Support" icon="message-square">
    If issues persist, reach out to our support team.
  </Step>
</Steps>

<Callout kind="info">
  Most upload issues are resolved by ensuring file quality and format compatibility.
</Callout>

const errorReport = {

fetch('/api/support/tickets', {

  ```python
import requests

error_report = {
  'error_type': 'upload_failed',
  'file_name': 'syllabus.pdf',
  'error_message': 'File processing timeout'
}

response = requests.post(
  '/api/support/tickets',
  json=error_report,
  headers={'Authorization': f'Bearer {api_key}'}
)
print(response.json())
  ```
