## Setting Up Integrations

<Steps>
  <Step title="Access Settings" icon="settings">
    Navigate to the integrations section in your account settings.
  </Step>
  <Step title="Choose Integration" icon="plus">
    Select the tool or platform you want to connect.
  </Step>
  <Step title="Authorize Access" icon="key">
    Grant DormWay permission to access your data.
  </Step>
  <Step title="Configure Sync" icon="refresh-cw">
    Set up automatic synchronization preferences.
  </Step>
  <Step title="Test Connection" icon="check">
    Verify the integration is working correctly.
  </Step>
</Steps>

<Callout kind="info">
  All integrations use secure OAuth authentication and encrypt your data.
</Callout>

<CodeGroup tabs="JavaScript,Python">
  ```javascript
const integrationConfig = {
  provider: 'google-calendar',
  syncFrequency: 'daily',
  includeReminders: true
};

fetch('/api/integrations', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'Authorization': `Bearer ${apiKey}`
  },
  body: JSON.stringify(integrationConfig)
});
  ```
  ```python
import requests

integration_config = {
  'provider': 'google-calendar',
  'sync_frequency': 'daily',
  'include_reminders': True
}

response = requests.post(
  '/api/integrations',
  json=integration_config,
  headers={'Authorization': f'Bearer {api_key}'}
)
print(response.json())
  ```
